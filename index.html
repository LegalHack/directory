Welcome to the Legal Hackers Prototype Directory of Law & Technology Educational Programs!

This is a project to create a distributed and shared public directory service listing events, programs and other resources relevant to legal hacking. The initial prototype is focused on law & technology educational program.

To see the current listing of law and technology educational programs, see the open directory listing page.

To make contributions to the directory, including proposing a new candidate programs for addition to the directory or letting us know about possible errors or other suggested improvements to the data or display, please use the input and feedback page.

To explore all the data in the open directory, including the data available at all the participating parts of the network some of which may not be available in the current curated version provided here, see the explore page.

To download a current version of all the data in the directory, please use the bulk download and data exports page.
